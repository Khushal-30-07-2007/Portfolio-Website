# 3D Interactive Portfolio

A cinematic personal portfolio website built with React, Three.js, and GSAP. This project focuses on high-end visual interactions, performance optimization, and creative frontend development.

## Features

- **Interactive 3D Elements**: Custom keyboard model with dynamic lighting and materials.
- **Scroll-Driven Animations**: Smooth transitions powered by GSAP ScrollTrigger.
- **Custom Shaders**: Unique visual effects using GLSL and React Three Fiber.
- **Responsive Design**: Optimized for performance across devices.

## Tech Stack

- **React** + **Vite**
- **Three.js** (@react-three/fiber, @react-three/drei)
- **GSAP** (GreenSock Animation Platform)
- **CSS Modules** for styling

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Khushal-30-07-2007/Portfolio-Website.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
